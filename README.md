# Basic Data Exploration and Cleaning using Pandas

## Project Overview

This project is part of the Celebal Internship assignment. The main objective of this assignment is to learn Python basics and perform basic data exploration and data cleaning using the Pandas library.

In this project, a shopping dataset is loaded into a Pandas DataFrame, explored using basic Pandas methods, cleaned by handling missing values and duplicates, and finally saved as a cleaned CSV file.

## Objective

The objective of this assignment is to:

1. Load a CSV dataset into a Pandas DataFrame.
2. Explore the dataset using head, tail, shape, columns, and data types.
3. Identify and handle missing values.
4. Perform basic operations like selecting columns and filtering rows.
5. Remove duplicate records.
6. Create a derived column named `total_amount`.
7. Save the cleaned dataset as a new CSV file.

## Dataset Used

The dataset used in this assignment is a shopping dataset from Kaggle.
