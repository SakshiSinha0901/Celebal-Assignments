# Basic Data Exploration and Cleaning using Pandas

This project performs basic data exploration and cleaning on a shopping dataset using Python and Pandas.

## Objective

The main objective is to learn Python basics and perform basic data exploration and data cleaning using Pandas.

## Dataset

Dataset used: Shopping Dataset from Kaggle.
